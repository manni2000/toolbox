# PDF Merge Tool - DailyTools247

Merge multiple PDF files into one document online for free.

## Features
- Combine unlimited PDF files
- Drag and drop interface
- Reorder pages before merging
- Fast processing
- No signup required
- Secure processing (files processed locally)

## How to use
1. Upload PDF files by clicking the upload area or dragging files
2. Reorder files by dragging them to desired positions
3. Click the "Merge PDFs" button
4. Download your merged PDF file
