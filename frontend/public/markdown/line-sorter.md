# Line Sorter Tool - DailyTools247

Sort text lines alphabetically or numerically.

## Features
- Alphabetical sorting
- Numerical sorting
- Reverse sorting
- Remove duplicates
- Case-sensitive options
- Copy to clipboard

## How to use
1. Paste your text lines
2. Select sorting options
3. Choose sort order
4. Click "Sort Lines" button
5. Copy sorted text
