# Image Crop Tool - DailyTools247

Crop images to specific areas with precision and ease.

## Features
- Visual crop selection
- Custom aspect ratios
- Precision cropping
- Multiple crop presets
- Real-time preview
- Batch cropping support

## How to use
1. Upload your image
2. Select crop area with mouse
3. Choose aspect ratio if needed
4. Adjust selection precisely
5. Click "Crop Image" button
6. Download cropped image

## Use cases
- Remove unwanted areas
- Create perfect squares
- Focus on specific subjects
- Social media formatting
- Print preparation
