# Text Diff Tool - DailyTools247

Compare two text files and highlight differences.

## Features
- Side-by-side comparison
- Difference highlighting
- Line-by-line analysis
- Merge suggestions
- Export differences
- Copy to clipboard

## How to use
1. Paste or upload first text
2. Paste or upload second text
3. Click "Compare" button
4. View highlighted differences
5. Export or copy results

## Use cases
- Compare document versions
- Review code changes
- Check text modifications
- Verify content updates
- Track changes
