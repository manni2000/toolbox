# PDF Rotate Tool - DailyTools247

Rotate PDF pages to correct orientation or improve readability.

## Features
- Rotate individual pages
- Rotate all pages at once
- 90, 180, 270 degree rotation
- Visual page preview
- Batch rotation support
- Maintain quality

## How to use
1. Upload your PDF file
2. Preview all pages
3. Select pages to rotate
4. Choose rotation angle (90, 180, 270)
5. Click "Rotate Pages" button
6. Download the rotated PDF

## Use cases
- Fix sideways scanned pages
- Correct document orientation
- Improve readability
- Standardize page orientation
- Fix mixed orientation documents
