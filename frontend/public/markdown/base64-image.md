# Base64 Image Converter - DailyTools247

Convert images to and from Base64 format for web development.

## Features
- Convert images to Base64
- Convert Base64 to images
- Support for all image formats
- Instant encoding/decoding
- Copy to clipboard
- Preview functionality

## How to use
1. Upload an image or paste Base64 code
2. Choose conversion direction
3. Click "Convert" button
4. Copy result or download file
5. Use in your projects
